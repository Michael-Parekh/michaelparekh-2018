michaelparekh.me



Objective: This project was created with the goal of displaying both my work and design techniques!

Duration: May 2018 - June 2018

Team: Michael Parekh

Tools: HTML, CSS, JavaScript, Particles.JS, SmoothScroll, ScrollTo, Bitmoji, Adobe Illustrator, Adobe Spark



The Start

I decided that I wanted to create a personal portfolio website to tell about myself, connect with other amazing people, and exhibit my skills.


Planning a Goal

I began by planning out the website. This included deciding what to add, choosing which tools would I use, and sketching out a timeline. I then decided to go with a "desktop-first" approach, which would mean that the website would be first designed on a desktop setting and then optimized for mobile devices. After this, I realized that I wanted the website to have a "modern" and "playful" feel to it, which contributed to using an interesting cartoon creation tool (Bitmoji) and choosing the unique typography.


The Process

To begin programming the project, I wrote out the HTML structure, which included separate sections for each part of the page. I then moved on to designing/styling using CSS, which was the most time-consuming part of the project. I wanted to convey my personality through the design, which was seen in things such as including my favorite color (blue, of course!). After this was complete, I used several interesting JavaScript libraries to add unique special effects. Once I had finished the desktop version of the website, I began to write media queries to optimize the page for mobile devices.



Closing Notes

Ultimately, this project was an extremely fun and fantastic way to start the summer. I am excited to now be able to publish my work here so that people can look at them and learn a bit about me!

- June 8, 2018
